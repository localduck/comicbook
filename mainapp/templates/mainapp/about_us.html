О проекте:

Небольшой проект на связке Python + Django, представляющий из себя Web Comic Reader. С перспективой развития в web comics aggregator, где у пользователя будет возможность получить доступ к комиксам через простой в использовании web UI: читать комиксы на сайте, создавать персонализированные коллекции, делится ими с другими пользователями.

На ресурсе не хранится ни одного материала. Весь материал взят с других ресурсов и размещен на сайте для ознакомления. Все размещенные на ресурсе материалы являются собственностью разместивших их пользователей до тех пор, пока не будет оснований считать иначе. Ресурс не несет ответственности за использование(как правомерное, так и неправомерное) третьими лицами материалов, размещенных на ресурсе.

Понимая актуальность соблюдения прав авторов хотим заверить Вас, что как только факт нарушения Ваших прав будет заранее установлен, мы предпримем все зависящие от нас меры для пресечения такого нарушения и, по возможности, постараемся посодействовать недопущению в дальнейшем подобных ситуаций.

Road Map:

  • Использование Google Drive, для хранилища загружаемых комиксов.
  • Добавить tags для комиксов и определить созависимые настройки пользователей.
  • Добавить юридическую информацию: условия использования, лицензионное соглашение, etc.
  • Добавить живой поиск: поиск по тегам, по названию, etc.
  • Открыть "роль" uploader для пользователей.
  • Настроить file downloader(Torrent-?).
  • Frontend: написать javascript comic reader app и добавить созависимые настройки пользователей.
  • Рефакторинг, оптимизация, unit-test кода.
